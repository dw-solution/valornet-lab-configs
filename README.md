## How to use these configs

1. Build the topology in **EVE-NG** or **GNS3** (diagrams in each lab folder)
2. Copy the relevant device config from `configs/` and paste into the router's CLI
3. Verify with the commands in the lab's `README.md`

All configs are tested in EVE-NG Community-CE against Cisco IOL / vIOS and Juniper vJunos routers.
